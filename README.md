## Get Started
- Clone repository
```
git clone https://github.com/carloscabir/fcc-project-timestamp-nodejs
```

- Move to project 
```cmd
cd project/name
```

- Install dependencies
```cmd
npm i
```

- Run project!!
```
npm start
```
